# conduit-node-client

Early-stage Node.js client for Conduit Events.

This package is under active development and is not production-ready.

Current focus:

- RabbitMQ transport
- schema validator
- envelope factory
- publish API
- subscribe API
- service queue naming rules
- routing conventions
- retry and dead-letter examples
- unit and integration testing

## Status

Experimental / work in progress.

APIs may change.
